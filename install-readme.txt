[color=blue][b][size=12pt][u]Introduction[/u][/size][/b][/color]
This modification adds several new admin maintenance functions including
[li]Recalculates user post counts[/li]
[li]Merge two users in to a single ID[/li]
[li]Mark inactive users as read all posts[/li]
[li]Recount forums daily statistics[/li]

This mod is compatible with SMF 2.0 Only.
